# mark-server-community

> 阉割过的mark-server(markpvp.kozow.com)服务端存档
> 仅有Kitsx, Nocheatplus插件以及参数

---

## 项目说明
此项目旨在让扣字大手子随随便便调整markpvp.kozow.com水晶服务器的ncp反作弊参数，请你fork，push，然后随随便便进行pr

---

clone仓库后，打开start.bat进行一个本地开服(jvm环境请手动配置)，然后用蹦蹦跳跳客户端随随便便进入0.0.0.0:25565，进行了ncp参数调整后，把config.yml随随便便commit到你的fork仓库，然后进行一个pull request耀武扬威。
